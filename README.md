# &#x20;             Osdag Bridge UI Backend



#### 🚀 Overview



Django REST API for managing states and districts.


#### 🛠 Tech Stack



* Python
* Django
* Django REST Framework
* SQLite



#### ⚙️ Setup

git clone <your-repo-link>
cd osdag-bridge-ui/backend
python -m venv venv
venv\\\\Scripts\\\\activate
pip install -r requirements.txt
python manage.py runserver


#### 🔗 APIs



* GET /api/states/
* GET /api/districts/<state\_id>/



#### 🔐 Admin



http://127.0.0.1:8000/admin/
